# senote.comサイトの構成

## 基本構成
- jaとenで多言語化
- トップに飛んできた場合jaへ
- 各ページ、ja/enを切り替えられるように
- 内容はペライチのページ＋別ページでコンタクト、プライバシーポリシー、運営者

## サイト内容（ペライチの内容）
- 日常生活や音楽に役立つ様々なツールやアプリ、情報を伝えるサイト
- コンテンツ
    - Pictmemo（iPhoneアプリ）https://apps.apple.com/jp/app/%E3%83%94%E3%82%AF%E3%83%88%E3%83%A1%E3%83%A2-%E5%86%99%E7%9C%9F%E3%81%A7%E3%83%A1%E3%83%A2%E3%81%99%E3%82%8B%E3%82%AB%E3%83%A1%E3%83%A9/id641415936
    - Sheet Music PDF Viewer（サイト）https://sheetmusicviewer.senote.com/
    - MusicTools by senote（サイト）https://musictools.senote.com/

各サイトにアクセスして文言と画像を取得してくる
サイト紹介文を書き、書くサイトにリンクを貼る


## 目的
MusicToolsにAdSenseを貼りたいが、そのためにまずsenote.comに必要な情報を載せる必要があるため。
